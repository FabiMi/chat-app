Fabis Chat App Fabis Chat App is a mobile application that allows users to chat with each other in real-time. This README provides an overview of the app and instructions for running and customizing it. Features User Registration: Users can enter their username and choose a background color before starting a chat session. Real-Time Chat: Once registered, users can join chat rooms and exchange messages in real-time with other participants. Background Color Selection: Users can choose from a selection of background colors to customize their chat interface.
